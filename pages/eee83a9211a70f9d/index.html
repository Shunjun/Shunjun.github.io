# webpack 构建

  • cross-env (opens new window) 运行跨平台设置和使用环境变量的脚本
  • *html-webpack-plugin (opens new window)自动生成 index.html，并且插入引用到的 bundle 和被拆分的 CSS 等资源路径。
  • hard-source-webpack-plugin (opens new window)给 modules 提供中间缓存步骤的 webpack 插件,(webpack 5 将内置此功能)
  • yargs (opens new window)Yargs 通过解析参数并生成一个优雅的用户界面，帮助你构建交互式命令行工具。
  • clean-webpack-plugin (opens new window)清理上次打包的 bundle,它会在第一次编译的时候删除 dist 目录中所有的文件，不过会保留 dist 文件夹，并且再每次 rebuild 的时候会删除所有不再被使用的文件。

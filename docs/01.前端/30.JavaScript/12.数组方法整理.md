### 创建新数组

返回的

| 方法名      | 描述                                                         | 参数                                                         | 返回值                                                       |
| :---------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **concat**  | 用于合并两个或多个数组                                       | ...arg:数组和/或值                                           | 合并后的数组                                                 |
| **entries** | 返回一个新的**Array Iterator**对象                           | 无                                                           | 包含原数组的[key,value]的迭代器对象                          |
| **keys**    | 返回一个包含数组中每个索引键的**Array Iterator**对象。       | 无                                                           | 包含key的迭代器对象                                          |
| **values**  | 返回一个包含数组中每个值的**Array Iterator**对象。           | 无                                                           | 新的 [`Array`](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array) 迭代对象。 |
| **filter**  | 创建一个新数组, 其包含通过所提供函数实现的测试的所有元素。   | **callback**(**element**[, **index**[, **array**]])[, **thisArg**] | 通过测试的元素组成的新数组                                   |
| **flat**    | 按照一个可指定的深度递归数组，打平指定深度的数组             | **depth**:结构深度，默认值为**1**                            | 被打平指定深度的数组                                         |
| **flatMap** | 首先使用映射函数映射每个元素，然后将结果压缩成一个新数组。它与 [map](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/map) 连着深度值为1的 [flat](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/flat) 几乎相同(**map**().**flat**(1))，但 `flatMap` 通常在合并成一种方法的效率稍微高一些。**flatMap** 能用于在map期间增删项目（也就是修改**items**的数量） | **callback**(**currentValue**[, **index**[, **array**]],**thisArg** | 一个新的数组，其中每个元素都是回调函数的结果，并且结构深度 **depth** 值为**1**。 |
| **map**     | 创建一个新数组，其结果是该数组中的每个元素是调用一次提供的函数后的返回值。 | **callback**(**currentValue**[, **index**[, **array**]],**thisArg** | 每个元素执行回调函数的结果组成的新数组                       |
| **slice**   | 返回一个由 `begin` 到 `end` (不包括**end**)的原数组的**浅拷贝**数组 | [**begin**:起始索引，默认为0，支持负索引[, **end**:结束索引，不包括end，默认为length-1，支持负索引]] |                                                              |



### 修改原数组

| 方法名         | 描述                                                         | 参数                                                         | 返回值                                                 |
| :------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------ |
| **copyWithin** | 浅复制数组的一部分到同一数组中的另一个位置                   | **target**:复制序列到该位置(支持负数索引),**start:**复制的起始位置(支持负数索引),**end**:复制的结束位置(支持负数，不包含该位置) | 修改后的数组                                           |
| **fill**       | 用一个固定值填充一个数组中从起始索引到终止索引内的全部元素。不包括终止索引。 | **value**:填充的值[, **start**:起始位置(支持负索引)[, **end**:结束位置(支持负索引)]] | 修改后的数组                                           |
| **pop**        | 从数组中删除最后一个元素，并返回该元素的值。                 | 无                                                           | 删除的元素，数组为空时返回**undefined**                |
| **push**       | 将一个或多个元素添加到数组的末尾，并返回该数组的新长度。     | **element** ***** **N** :被添加到数组末尾的元素。            | 新的**length**                                         |
| **reverse**    | 将数组中元素的位置颠倒，并返回该数组。                       | 无                                                           | 颠倒后的数组。                                         |
| **shift**      | 从数组中删除**第一个**元素，并返回该元素的值。               | 无                                                           | 删除的元素，数组为空时返回**undefined**                |
| **sort**       | 对数组**原地**排序，并返回数组。                             | **compareFunction**(**firstEl**,**secondEl**),返回值如果大于0则交换顺序 | 排序后的数组。请注意，数组已原地排序，并且不进行复制。 |
| **splice**     | 通过删除或替换现有元素或者原地添加新的元素来修改数组,并以数组形式返回被修改的内容。 | **start**[, **deleteCount**[, **item1**[, **item2**[, ...]]]] | 被删除的元素组成的一个数组。                           |
| **unshift**    | 将一个或多个元素添加到数组的**开头**，并返回该数组的**新长度** | **element** ***** **N** :被添加到数组开头的元素。            | 新的**length**                                         |



### 不修改也不创建数组

| 方法名             | 描述                                                         | 参数                                                         | 返回值                                                 |
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------ |
| **every**          | 测试数组内的所有元素是否都能通过某个指定函数的测试           | **callback**(**element**,**index**,**array**),**thisArg**:callback的this值 | 布尔值，如果为空数组 始终为**true**                    |
| **some**           | 测试数组中是不是至少有1个元素通过了被提供的函数测试。        | **callback**(**element**,**index**,**array**),**thisArg**:callback的this值 | 布尔值，如果为空数组 始终为**false**                   |
| **find**           | 返回数组中满足提供的测试函数的第一个元素的值,否则返回**undefined** | **callback**(**element**,**index**,**array**),**thisArg**    | 满足测试条件的第一个值                                 |
| **findIndex**      | 返回数组中满足提供的测试函数的第一个元素的**索引**。否则返回**-1** | **callback**(**element**,**index**,**array**),**thisArg**    | 通过提供测试函数的第一个元素的**索引**，否则返回**-1** |
| **forEach**        | 对数组的每个元素执行一次给定的函数。                         | **callback**(**currentValue**[, **index**[, **array**]],**thisArg** | **undefined**                                          |
| **includes**       | 判断一个数组是否包含一个指定的值                             | **valueToFind**:要查找的值[, **fromIndex**:从索引处开始查找(支持负索引)] | 布尔值                                                 |
| **indexOf**        | 返回在数组中可以找到一个给定元素的第一个索引,如果不存在，则返回-1。 | **searchElement**:要查找的值[, **fromIndex**:从索引处开始查找(支持负索引)] | 索引值，不存在返回**-1**                               |
| **lastIndexOf**    | 返回指定元素在数组中的最后一个的索引，如果不存在则返回 -1。从 **fromIndex**向前查找 | **searchElement**:要查找的值[, **fromIndex**:从索引处开始向前查找(支持负索引)] | 元素最后一次出现的索引                                 |
| **join**           | 一个数组（或一个[类数组对象](https://developer.mozilla.org/zh-CN_docs/Web/JavaScript/Guide/Indexed_collections#working_with_array-like_objects)）的所有元素连接成一个字符串并返回这个字符串。尾项不添加分隔符 | **separator**:分隔符，默认为"**,**"                          | 拼接的字符串,数组为空返回空字符                        |
| **reduce**         | 对数组中的每个元素执行**reducer**函数(升序执行)，将其结果汇总为单个返回值。 | **callback**(**accumulator**:累计器,上一轮的返回值或者初始值, **currentValue**[, **index**[, **array**]])[, **initialValue**:初始值] | 累计器的值                                             |
| **reduceRight**    | 与**reduce**类似，但是从数组末尾开始遍历                     | **callback**(**accumulator**:累计器,上一轮的返回值或者初始值, **currentValue**[, **index**[, **array**]])[, **initialValue**:初始值] | 累计器的值                                             |
| **toLocaleString** | 返回一个字符串表示数组中的元素。数组中的元素将使用各自的 `toLocaleString` 方法转成字符串 | [locales[,options]                                           | 表示数组元素的字符串                                   |
| **toString**       | 一个字符串，表示指定的数组及其元素。用逗号分隔的每个数组元素。 | 无                                                           | 一个表示指定的数组及其元素的字符串。                   |



### 静态方法

| 方法名      | 描述                                                         | 参数                                                         | 返回值                                                       |
| ----------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |

| **isArray** | 确定传递的值是否是一个 [`Array`](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array)。 | **obj**:要检测的值                                           | 布尔值                                                       |
| **of**      | 创建一个具有可变数量参数的新数组实例，而不考虑参数的数量或类型。 | **element** ***** **N**:任意个参数，将按顺序成为返回数组中的元素。 | 新的 [`Array`](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array) 实例 |
